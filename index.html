Stack Lang

A simple yet powerful stack-based programming language

Quick Reference

Stack Operations

push n Push number n onto stack dup Duplicate top value swap Swap top two values drop Remove top value

Arithmetic

+ Add top two values - Subtract (a b → a-b) * Multiply / Divide (a b → a/b)

Comparison

= Equal to (returns 1 or 0) < Less than > Greater than

Functions

def name: Define function name Call function by name 
def double:
    dup
    +

def square:
    dup
    *

Control Flow

if: If top of stack is 1 (true) else: Optional else block 
# Check if number is positive
push 5
dup
push 0
>
if:
    println  # Outputs: 5
else:
    drop
    push 0
    println

Output

println Print top value with newline print Print top value without newline 
# Print numbers
push 5
println
push 10
print

Examples

# Calculate (3 + 4) * 2
push 3
push 4
+     # Stack now has 7
push 2
*     # Stack now has 14
println  # Outputs 14

# Compare numbers
push 5
push 3
>     # Returns 1 (true)
println
